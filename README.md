# What is this?
Tools to benchmark the Linux framebuffer (`/dev/fb0`) using CPU / software-renderer.

# How to compile ?
* Compile: `make`
* Static compile: `make LDFLAGS="-static"`
* Install: `sudo make install`
* Uninstall: `sudo make uninstall`
